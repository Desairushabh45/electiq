# 🗳️ ElectIQ — Your Indian Election Education Assistant

> An interactive AI-powered platform to help every Indian 
> citizen understand the election process, timelines, and 
> democratic steps in a simple and engaging way.

## 🎯 Problem Statement
Create an assistant that helps users understand the election 
process, timelines, and steps in an interactive and 
easy-to-follow way.

ElectIQ solves this by providing:
- Step-by-step Indian election timeline
- AI-powered chat assistant (powered by Groq + Llama3)
- Interactive quiz to test election knowledge
- Complete ECI information hub
- Hindi and English language support

## ✨ Features
- 🗺️ Election Timeline — All 8 stages from announcement to oath
- 🤖 AI Assistant — Ask anything about Indian elections
- 📝 Interactive Quiz — Test your election knowledge
- 🏛️ About ECI — Election Commission of India info hub
- ⚙️ How It Works — EVMs, VVPAT, voter registration explained
- 🇮🇳 Bilingual — Hindi and English support

## 🛠️ Tech Stack
- React + Vite
- Tailwind CSS
- Groq API (Llama3-8b) — AI Assistant
- Firebase Hosting — Deployment
- Firebase Analytics — Usage tracking
- Google Antigravity — AI-native development

## 🏃 How to Run Locally
1. Clone the repo
2. Run npm install
3. Create .env file with your API keys (see .env.example)
4. Run npm run dev
5. Open http://localhost:5173

## 🔑 Environment Variables
VITE_GROQ_API_KEY=your_groq_api_key
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_PROJECT_ID=electiq-ffd78
