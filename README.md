# 🗳️ ElectIQ — Interactive Election Process Education Assistant

## Problem Statement
"Create an assistant that helps users understand the 
election process, timelines, and steps in an interactive 
and easy-to-follow way."

## How ElectIQ Solves This
ElectIQ directly addresses this by providing:
✅ Interactive 8-stage election timeline
✅ Step-by-step process explanations  
✅ AI assistant for instant answers
✅ Quiz to verify understanding
✅ Visual and bilingual content (Hindi + English)
✅ Mobile-first responsive design

## ✨ Key Features

### 🤖 AI Election Assistant (Core Feature)
- Powered by Google Gemini AI
- Answers any question about Indian elections
- Responds in Hindi and English
- Covers ECI, EVMs, VVPAT, voter registration, MCC

### 🗓️ Interactive Election Timeline
- 8 visual stages of Indian election process:
  1. Election Announcement & Model Code of Conduct
  2. Voter Registration Deadline
  3. Nomination Filing by Candidates
  4. Campaign Period
  5. Voting Day (EVM + VVPAT)
  6. Vote Counting Process
  7. Results Declaration
  8. Winner Takes Oath of Office
- Click each stage for detailed explanation

### 📚 Step-by-Step Process Guides
- How to Register to Vote (Form 6 process)
- How EVM machines work
- How VVPAT verifies your vote
- How votes are counted in India

### 📝 Interactive Quiz
- 6 multiple choice questions
- Tests knowledge of election process
- Shows correct answers with explanations
- Score display with performance feedback

### 🏛️ About ECI Hub
- Election Commission of India facts
- Article 324 explanation
- Role of Chief Election Commissioner
- How ECI ensures fair elections

## 🛠️ Tech Stack
- React + Vite — Frontend framework

## 🏃 How to Run Locally

### Prerequisites
- Node.js 18+
- npm or yarn

### Setup
1. Clone the repository
   git clone https://github.com/Desairushabh45/electiq.git
   cd electiq

2. Install dependencies
   npm install

3. Create .env file
   cp .env.example .env
   Add your API keys to .env

4. Run development server
   npm run dev

5. Open browser at http://localhost:5173

### Run Tests
   npm test

### Build for Production
   npm run build
